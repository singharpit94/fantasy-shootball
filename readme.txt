FANTASY SHOOTBALL
=================


FANTASY SHOOTBALL for MUKTI'15 on PHP

Pre-Requirements
1. LAMP/WAMP/XAMPP server.

Installation
1. Install LAMP/WAMP/XAMPP.
2. Import the project to respective server location
3. Import the database sas given in the .sql format (sas.sql) to phpmyadmin.
4. Check the .\include\const.php file. Change the Data Source/Server from 'localhost' to your server name
   	Simlarly for User ID/US_NAME, Password/PASS_W.
5. Run server and open index.php.
